## Getting Started

```bash
# Clone repo
git clone https://github.com/wikaex/landing-page-airdrop.git
cd landing-page-airdrop

# Install dependencies
npm install

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
